## TMX Free Candy

Public-facing utility scripts, helpers, and lightweight tooling from Terramax.

This site is a simple index for selected public releases, reference materials, and small operational tools made available as-is.

## Available Resources

- KDE-Services bootstrap / dependency helper
- Shell and automation utilities
- Small deployment and environment helpers
- Reference notes and supporting documentation

## Notes

These materials are provided for general use, testing, and adaptation.  
They are shared in a practical working form and may evolve over time.

Please review all scripts before use in a production environment.
